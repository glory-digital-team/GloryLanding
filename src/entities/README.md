# entities — слой бизнес-сущностей

Здесь живут **сущности** предметной области (например: `user`, `product`, `article`).
Сущность описывает «что это» — её данные, типы и базовое отображение.

## Правила

- Может импортировать **только** из `shared`.
- Не может импортировать из `features`, `widgets`, `views`, `app` и из других `entities`.
- Каждая сущность — это slice со своим публичным API (`index.ts`).

## Структура среза

```
entities/user/
├── ui/            # представление сущности (напр. UserCard, UserAvatar)
├── model/         # типы, схемы, селекторы
├── api/           # запросы к бэкенду (по необходимости)
└── index.ts       # публичный API
```
